# FullStack Tasks - Frontend

React 18 + Vite + TypeScript + Tailwind + Shadcn/ui dashboard for the Task management API.

## Scripts

- `npm run dev` - start dev server (http://localhost:5173, proxies `/api` and `/actuator` to `http://localhost:8080`)
- `npm run build` - typecheck and build production bundle
- `npm run preview` - preview built bundle
- `npm run test` - run unit/component tests once
- `npm run test:watch` - watch mode tests
- `npm run typecheck` - tsc only
- `npm run lint` - eslint
